MVC?

  • Model - the application data, or state, typically kept in a database
  • View - the user interface
  • Controller - the application brain that determines what the user can do given the state of the application

MVC

  • A design pattern , a software architecture , not a hard-coded thing
  • Web app frameworks that provide an MVC-like structure out of the box:
    • Ruby on Rails (Ruby)
    • Laravel (PHP)
    • Django (Python)
    • Angular (JavaScript)
