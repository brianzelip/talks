git

  • created to manage the work of hundreds of people around the world
  • "version control system" for tracking changes in computer files over time
  • access any version of any file at any time
  • allows multiple people to collaborate on the same files
  • each collaborator has a complete copy of the project repository, independent of network access or a central server
